# yogiyo-front

## ✏️ commit convention

| 유형     | 설명                                                  |
| -------- | ----------------------------------------------------- |
| feat     | 새로운 기능 추가                                      |
| fix      | 버그 수정 또는 오타 수정                              |
| refactor | 코드 리팩토링                                         |
| design   | 사용자 인터페이스 디자인 변경, CSS 수정 등            |
| comment  | 주석 추가 또는 변경                                   |
| style    | 코드 포맷팅, 세미콜론 누락, 코드 변경 없음            |
| test     | 테스트 코드 추가, 수정, 삭제, 비즈니스 로직 변경 없음 |
| chore    | 위의 유형에 해당하지 않는 기타 변경사항               |
| init     | 프로젝트 초기 생성                                    |
| rename   | 파일 또는 폴더명 변경 또는 이동                       |
| remove   | 파일 삭제 또는 관련 작업만 수행                       |
